# Developer Portfolio

A modern, minimalist personal portfolio and blog site built with Next.js and TailwindCSS.

## Features

- Responsive design
- About Me section
- Articles/Blog
- Contact form
- Dark/Light mode theme

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Tech Stack

- Next.js
- React
- TypeScript
- TailwindCSS
